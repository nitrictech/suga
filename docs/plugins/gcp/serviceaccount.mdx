---
title: Service Account
description: Service account for secure authentication and authorization in Google Cloud
---

## Configuration

<ParamField path="project_id" type="string">
  Google Cloud Project ID
  
  **Example:** `my-project-123`
</ParamField>

<ParamField path="trusted_actions" type="list(string)">
  List of trusted actions for the service account
  
  **Example:** `["iam.serviceAccounts.actAs"]`
</ParamField>
